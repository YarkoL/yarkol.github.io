blank loads the linked file in a new, unnamed browser window. _parent loads the linked file in the parent frameset or window of the frame that contains the link. If the frame containing the link is not nested, the linked file loads into the full browser window. _self loads the linked file in the same frame or window as the link. This target is implied, so you generally don’t need to specify it. _top loads the linked file in the full browser window, thereby removing all frames.
